🎉 **We're thrilled to announce the launch of the PVK Console project blog!**

This ambitious project aims to create an open-source desktop replica of the legendary PVKK planetary defense console. Our mission is to build a fully functioning version that not only looks authentic but captures the spirit and interactive experience of the original.

---

## 🎯 Project Vision

The PVK Console represents more than just a replica – it's a tribute to innovative design and interactive technology. We're committed to:

- ✨ **Authenticity**: Staying true to the original design while adding modern improvements
- 🔓 **Open Source**: Making all designs, code, and documentation freely available
- 🌍 **Community-Driven**: Building this together with passionate contributors worldwide
- 🎓 **Educational**: Providing detailed documentation for learning and experimentation

## Current Status
We have successfully completed the first prototype PCB. We are finalizing the firmware and adding the last features. The desktop application is just for testing purposes, but we will be adding more features and improving the UI in the coming weeks. We are also working on the documentation as well as the schematics and PCB design. We are positive that we will be able to release the first mod version of the console shortly after the game is released.
